Control de versiones

versiones locales

Los controladores locales copian los archivos diractamente a otro directorio pero no sabes en que directorio te encuentras y solo pueden ser copiados o movidos dentro de un sistema cerrado.

versiones centralizadas

Los controladores centralizados permiten la conexión entre un sistema A y el servidor y al sistema b le permite acceder a esos datos tambien.

controladores distribuidos

Los controladores distribuidos permiten tanto la conexión entre servidor y computadora como computadora con computadora.

Qué es GitHub y cuáles son sus utilidades

GitHub es un controlador de versiones centralizado y permite tanto el subir tu codigo y modificarlo o actualizarlo como llevar un control de las diferentes versiones que has subido y así no perder el codigo anterior.
